# 🎮 Minecraft UI Builder

Визуальный конструктор GUI меню для Minecraft модов/плагинов. Создавайте структуру меню на основе сундуков, настраивайте кнопки, привязывайте предметы из инвентаря игрока как триггеры — и тестируйте навигацию прямо в браузере.

## ✨ Возможности

- 🎨 **Визуальный редактор** — настройка слотов в окне сундука (1–6 строк) с настоящими текстурами Minecraft
- 📦 **Библиотека 1400+ предметов** — все стандартные предметы и блоки Minecraft с категориями и поиском
- 🎒 **Триггеры из инвентаря** — настраивайте какой предмет в каком слоте инвентаря игрока открывает какое меню
- 🔗 **Граф навигации** — визуализация связей между меню
- 🎮 **Test Mode** — эмуляция как в игре: клик по предмету → открытие меню, навигация Open / Back / Close
- 🌐 **i18n** — интерфейс на русском и английском
- 📥 **Export / Import JSON** — конфиги передаются разработчику для имплементации в моде
- 💾 **Auto-save** в localStorage

## 🚀 Запуск локально

```bash
npm install
npm run dev
```

Открыть в браузере: `http://localhost:5173/`

## 📦 Сборка

```bash
npm run build
```

Готовая сборка в папке `dist/` — можно хостить где угодно.

## 🛠 Стек

- React 18 + Vite
- Vanilla CSS
- Текстуры из ресурспака Minecraft 26.1.2

## 📄 Workflow с командой

1. **Дизайнер/менеджер** создаёт меню в инструменте
2. **📥 Export JSON** → передаёт файл `minecraft-ui-config.json`
3. **Разработчик** открывает тот же инструмент → **📤 Import JSON** → видит схему → пишет код мода
