# 🎮 Minecraft UI Builder

Визуальный конструктор GUI меню для Minecraft на базе сундуков. Хех)

Живая версия: **https://dkotsergeev.github.io/minecraft-ui-builder/**

## 🚀 Запуск локально

```bash
npm install
npm run dev
```

Открыть в браузере: `http://localhost:5173/`

## 📦 Сборка

```bash
npm run build
```

Готовая сборка в папке `dist/` — можно хостить где угодно.

## 🚢 Деплой

`./deploy.sh "что изменил"` — закоммитит локальные правки и запушит в `main`.
GitHub Actions сам соберёт и выложит обновление на Pages за ~1 минуту.

## 🛠 Стек

- React 18 + Vite
- Vanilla CSS
- Текстуры из ресурспака Minecraft 26.1.2

## 📄 Как передать что ты там нашаманил

1. **Дизайнер/менеджер** создаёт меню в инструменте
2. **📥 Export JSON** → передаёт файл `minecraft-ui-config.json`
3. **Разработчик** открывает тот же инструмент → **📤 Import JSON** → видит схему → пишет код мода

## 🗿 Как устроен каталог голов

Внутри лежит индекс на ~92 000 голов — скачан с minecraft-heads.com через их публичный
API и сохранён в `public/heads-catalog.json` (≈9 МБ, после gzip отдаётся как ≈4 МБ).
Это всё категории сразу: alphabet, animals, blocks, decoration, food-drinks, humans,
humanoid, monsters, plants, miscellaneous.

Каждая запись это `{ n: имя, h: texture hash, t: категория }`. Поиск гоняется
прямо в браузере по этим двум полям, так что фильтрация мгновенная даже на 92k
записей. Картинка собирается на лету через `https://mc-heads.net/head/<hash>/64`
— тот же 3D-угол, как у головы-предмета в игре. Прямые ссылки с
minecraft-heads.com мы намеренно не используем: Cloudflare на их CDN режет
hot-linking и просто отдаёт 403.

Если каталога мало — внизу пикера есть поле «Голова игрока по нику». Вводишь
ник Minecraft-аккаунта (например `DeloriaZ`), жмёшь «Применить» — слот получает
URL `https://mc-heads.net/head/DeloriaZ/64`, который рендерится тем же сервисом.
Удобно когда нужна именно твоя голова или голова конкретного игрока без рытья
по каталогу.

### Обновить каталог

Скрипт фетча лежит в `scripts/fetch_heads.py`. Запускаешь — он перекачает все
категории с minecraft-heads.com и перезапишет `public/heads-catalog.json`.
Категория растёт сама собой, никаких ручных правок не нужно.
